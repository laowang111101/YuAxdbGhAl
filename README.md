# 图书管理系统 python187

## 项目概述

图书管理系统是基于Python语言开发的一个课程设计项目，旨在为图书馆提供一个便捷、高效的管理工具。

## 技术栈

- **后端**: Python 3.x, Flask
- **数据库**: SQLite
- **前端**: HTML, CSS, JavaScript

## 功能模块

1. 用户管理
   - 用户注册
   - 用户登录
   - 用户信息修改
   - 用户权限控制
2. 图书管理
   - 图书信息录入
   - 图书检索
   - 图书借阅
   - 图书归还
3. 系统管理
   - 数据统计
   - 借阅记录查询

## 系统角色

- 系统管理员：负责系统的维护与用户管理
- 普通用户：进行图书借阅与归还操作

## 运行环境

- 操作系统：兼容Windows/Linux/MacOS
- Python版本：3.x
- 所需依赖：Flask, SQLite

## 部署步骤

1. 克隆项目代码到本地
2. 安装依赖项 `pip install -r requirements.txt`
3. 初始化数据库 `python manage.py db init`
4. 运行应用 `python manage.py runserver`

## 核心亮点

- **简洁的UI设计**：便于用户快速上手操作
- **灵活的角色权限管理**：根据不同角色赋予不同权限，确保系统安全
- **易扩展的数据模型**：方便后续根据需求增加新功能
- **轻量级技术栈**：基于Flask，易于部署和维护
- **完整的借阅流程**：覆盖图书从入库到借阅的全流程管理
